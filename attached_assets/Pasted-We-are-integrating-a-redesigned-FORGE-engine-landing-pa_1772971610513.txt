We are integrating a redesigned FORGE engine landing page into TIBER. The goal is to make the FORGE landing page feel native to the product, not like a generic marketing page. It should present FORGE as the scoring backbone of TIBER and visually explain how the engine works.

Objective:
Replace or redesign the current FORGE landing page so it has a darker premium-tech look and a schematic-style engine visualization that feels like part of the actual TIBER product.

Design intent:
The page should communicate that FORGE is the machine behind the grade. It ingests raw football data, normalizes it, blends weighted pillars, and outputs an Alpha score that powers rankings, player pages, trade tools, and agent responses. The page should feel like exposed product architecture, not dashboard fluff.

Implementation requirements:

Use the existing TIBER app shell, navigation, spacing, and styling conventions so the page fits naturally with the rest of the app.

Keep the page dark, clean, and technical.

Avoid generic SaaS marketing styling.

Preserve responsiveness for desktop and mobile.

Use existing design tokens/components where possible.
