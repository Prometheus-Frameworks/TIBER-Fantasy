## üí° Key Design Principles

1. **Data First, Philosophy Second**
   - Weekly data at RAG position 1
   - Brain OS philosophy at position 3+

2. **Interpret, Don't Dump**  
   - Max 2 numbers in opening
   - Plain language required
   - Brain OS lens applied

3. **Trust but Verify**
   - Trust data blocks as ground truth
   - Validate through comprehensive testing

4. **Graceful Degradation**
   - Say "I don't have that data" when unavailable
   - Offer general analysis as alternative

5. **Season-Agnostic**
   - Built for 2024, ready for 2025
   - Zero code changes needed to switch
