### ✅ **4. Depth-Chart Filter (pseudocode → production-ready)**

```python
def fantasy_relevant_depth(df_weekly):
    caps = {'WR': 4, 'RB': 3, 'TE': 2}
    return (
        df_weekly
        .query("snap_share >= 0.20")
        .sort_values(['team', 'pos', 'season', 'week', 'snap_share'],
                     ascending=[True, True, True, True, False])
        .groupby(['team', 'pos', 'season'])
        .head(caps.get(df.name[1], 4))
    )
```

- Runs **after every week’s data load**.  
- UI only shows players passing the filter.  
- Greyed-out section lists “inactive / <20 % snap” for transparency.
