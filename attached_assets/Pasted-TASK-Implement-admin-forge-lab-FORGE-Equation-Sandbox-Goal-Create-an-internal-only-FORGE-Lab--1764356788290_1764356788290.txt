2. Equation Registry

Create a small “equation registry” in a shared file, e.g.:

client/src/forgeLab/equations.ts

Each equation should define:

export type ForgeLabInputDef = {
  key: string;
  label: string;
  min: number;
  max: number;
  step: number;
  defaultValue: number;
};

export type ForgeLabEquation = {
  id: string;           // "TD_ENVIRONMENT_BOOST"
  name: string;         // "TD Environment Boost"
  description: string;  // short human-readable summary
  inputs: ForgeLabInputDef[];
  compute: (inputs: Record<string, number>) => {
    outputs: Record<string, number>;
    steps: string[]; // human-readable steps
  };
};

Implement at least one equation to start:

The compute function should:
	•	Calculate:
	•	p_success
	•	z
	•	E
	•	TD_alpha_star
	•	Return:
	•	outputs:
	•	{ p_success, z, E, TD_alpha_star }
	•	steps:
	•	Array of strings describing the calculation step-by-step
	•	Example:
	•	"p_success = p_OL * p_QB * p_Tgt * p_Sep * p_Catch = 0.0614"
